# Apple Global Sales Analysis 2020–2024

## Overview
Exploratory data analysis of Apple's global sales data across 75 stores, 
89 products, and 10 product categories. The project covers data exploration 
using SQL and visualization using Tableau.

## 🛠 Tools Used
- **MySQL** — data querying and analysis
- **Tableau** — interactive dashboard

## 📂 Dataset
- `sales.csv` — transaction records (sale_id, sale_date, store_id, product_id, quantity)
- `products.csv` — product details (product_id, product_name, category_id, launch_date, price)
- `category.csv` — product categories (category_id, category_name)
- `stores.csv` — store locations (store_id, store_name, city, country)

## 🔍 Key Questions Answered
1. What are the top 5 products by units sold?
2. What are the top 5 products per category?
3. Which stores rank in the top 10 by revenue and units sold?
4. What are total sales by country?
5. How does revenue trend month over month?
6. Which price range performs best?
7. What is each category's % contribution to total revenue?

## 💡 Key Findings
- Total Revenue: $549B across 5.7M units sold
- HomePod mini was the top selling product with 128,870 units
- Accessories generated the highest category revenue
- Apple Chadstone was the top store by units sold
